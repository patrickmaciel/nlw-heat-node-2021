# Comandos utilizados

```sh
yarn init -y
yarn add express
yarn add @types/express typescript ts-node-dev
```

Edita o `package.json` e adiciona a linha dos scripts

```js
"scripts": {
    "dev": "ts-node-dev src/app.ts"
}
```

```
yarn add prisma -D
yarn prisma init
yarn add dotenv
yarn add axios @types/axios
yarn add jsonwebtoken
```
